Primeiros passos:

Os primeiros passos para construir uma página na web é através do uso do html.
A parte textual fica toda a cargo de p para parágrafo e br para quebra de página.

Para quebra temática, usa-se o hr 

 pre é um elemento que define um texto pré formatado, respeitando os espaços e as quabras de linha no momento que estamos escrevendo para ser exibido.
        Se pular uma linha no código, na página também aparecerá da mesma forma.
b é usado para destacar um texto em negrito já o elemento strong é um elemento que traz uma relevância forte a palavra ou texto. Para um texto itálico utiliza-se o i.
Já o em enfatiza a importância semântica de um texto. Para um texto com letras maiores, usa-se o big. Percebe que aumentou? Agora, para diminuir o texto, usa-se small.
Já o sup eu vou testar porque eu não sei a funcionalidade dele. E continuo sem saber kkkk. Mas aí vem o que por definição é um elemento que define um texto subscrito. Muito bom para legendas de imagens.
